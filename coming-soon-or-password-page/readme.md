# Coming Soon / Password Page

A free Coming Soon or Password Page section for Shopify themes that you can use on any theme, free or paid, to create a compelling pre-launch landing page.

## Overview

The Coming Soon / Password Page section transforms your Shopify store's default password page into a branded, professional pre-launch experience. Display a countdown timer, email signup, and brand messaging to build anticipation and capture leads before your store goes live.

## Features

- Custom Coming Soon layout for the Shopify Password Page
- Compatible with any Shopify theme (Free or Paid)
- Optional countdown timer to your launch date
- Email signup / newsletter capture
- Fully customizable headline, description, and branding
- Responsive design for all screen sizes
- No paid app required

## Getting Started

1. In your Shopify admin, navigate to **Online Store → Themes → Edit Code**
2. Under `sections/`, create a new section and paste the provided Liquid code
3. Navigate to **Online Store → Themes → Customize → Password Page**
4. Add the new **Coming Soon** section to the Password Page template
5. Configure your launch messaging and optional countdown date
6. Save and publish your changes

## Customization

- **Headline & description:** Set your pre-launch messaging and brand story
- **Logo:** Display your store logo prominently
- **Countdown timer:** Set your target launch date to show a live countdown
- **Background:** Configure a background image or color
- **Email signup:** Enable newsletter capture to collect leads before launch
- **Social links:** Add links to your social media profiles
