# Shape Divider Section

Add a Shape Divider section to your Shopify store with no coding required — a free section that creates decorative wave, curve, or geometric transitions between page sections.

## Overview

The Shape Divider section adds visually appealing SVG shape dividers between content sections on your Shopify storefront. These decorative transitions — including waves, angles, curves, and zigzags — create a polished, layered page design that makes your store look more professional and visually cohesive.

## Features

- Multiple SVG shape divider styles (wave, curve, angle, zigzag, and more)
- No coding required — configure everything in the theme customizer
- Compatible with any Shopify theme (Free or Paid)
- Smooth, crisp SVG rendering at any screen size
- Configurable shape color to match adjacent sections
- Top and bottom divider placement options
- Fully responsive for all screen sizes

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Shape Divider** section between two content sections on any page
4. Select your preferred divider shape and configure the color
5. Save and publish your changes

## Customization

- **Shape style:** Choose from wave, curve, angle, tilt, zigzag, or custom SVG shapes
- **Color:** Set the divider color to match the section above or below
- **Height:** Adjust the vertical size of the divider
- **Flip:** Mirror the shape horizontally or vertically
- **Placement:** Use as a top or bottom border for any content section
