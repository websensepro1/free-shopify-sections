# Circle Menu

A Circle Menu section for Shopify that works with all themes, allowing you to easily add icon-based circular navigation to your storefront without any additional paid apps.

## Overview

The Circle Menu section adds a visually distinctive circular navigation element to your Shopify store. Each circle displays an icon or image with a label, linking to a collection, page, or product. This is an excellent way to present top-level categories or featured collections in an engaging, visual format.

## Features

- Circular icon/image navigation tiles
- Compatible with all Shopify themes (Free or Paid)
- No paid app required
- Configurable number of circles per row
- Each circle links to a custom URL
- Supports icons, images, and text labels
- Fully responsive for mobile and desktop
- Easy to manage via the theme customizer

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Circle Menu** section to your desired page template
4. Add circle items (icon/image, label, link) using the block settings
5. Save and publish your changes

## Customization

- **Circle items:** Add or remove menu circles via the theme customizer
- **Icons or images:** Upload a custom image for each circle
- **Labels:** Set display text shown beneath each circle
- **Links:** Configure the destination URL for each circle item
- **Size & spacing:** Adjust circle diameter and gap between items
- **Hover effects:** Customize animation on hover
