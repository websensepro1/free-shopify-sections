# Animated Banner

A free Animated Banner section for Shopify themes that you can use on any theme, free or paid, to create visually engaging hero and promotional banners.

## Overview

The Animated Banner section brings your Shopify storefront to life with smooth CSS animations applied to banner images and text. This section is designed to work seamlessly with any Shopify theme and requires no coding knowledge to set up — simply install and customize through the theme editor.

## Features

- Compatible with any Shopify theme (Free or Paid)
- Smooth CSS entrance and loop animations
- Customizable banner image, headline, and subtext
- Call-to-action button with configurable link
- Fully responsive layout for mobile and desktop
- Lightweight implementation with no external dependencies
- Easy setup through the Shopify theme customizer

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Animated Banner** section to your desired page template
4. Upload your banner image and enter your heading and subtext
5. Configure the CTA button link and label
6. Save and publish your changes

## Customization

- **Banner image:** Upload a high-resolution background or feature image
- **Headline & subtext:** Set your main message and supporting copy
- **Animation style:** Adjust CSS animation type, duration, and delay
- **CTA button:** Configure button text, link, and color
- **Overlay:** Add a color overlay with adjustable opacity for text readability
- **Height:** Control the banner height for desktop and mobile breakpoints
