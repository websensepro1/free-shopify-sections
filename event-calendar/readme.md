# Event Calendar

Add a fully functional event calendar to your Shopify store for free, without using any apps, to keep customers informed about upcoming events and promotions.

## Overview

The Event Calendar section integrates a clean, interactive calendar display into your Shopify storefront. Use it to showcase upcoming sales, product launches, workshops, live events, or any date-based content. The implementation is app-free and works on any Shopify theme.

## Features

- Fully functional event calendar display
- No app required — built with Liquid and JavaScript
- Works on any Shopify theme (Free or Paid)
- Add, edit, and remove events via the theme customizer
- Display event titles, dates, descriptions, and links
- Monthly navigation with previous/next controls
- Fully responsive for all screen sizes

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Event Calendar** section to your desired page template
4. Add event blocks with date, title, description, and optional link
5. Save and publish your changes

## Customization

- **Events:** Add or remove event blocks via the theme customizer
- **Event details:** Set date, title, short description, and CTA link per event
- **Color coding:** Assign different colors to event types or categories
- **Calendar view:** Configure month display and navigation behavior
- **Section heading:** Add an optional title above the calendar
