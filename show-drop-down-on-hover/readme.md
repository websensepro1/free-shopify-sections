# Show Drop Down on Hover

Learn how to easily enable a hover effect for your Shopify Dawn theme dropdown menu using a simple code snippet, improving navigation usability.

## Overview

By default, the Shopify Dawn theme requires a click to open dropdown navigation menus. This guide shows you how to modify the theme with a small code snippet so that dropdown menus open automatically on hover — providing a faster, more intuitive navigation experience for desktop users.

## Features

- Dropdown menus open on hover (no click required)
- Compatible with Shopify Dawn theme
- Simple code snippet — no full theme override needed
- Improves navigation speed and user experience on desktop
- Falls back gracefully to click behavior on mobile/touch devices
- No paid app required

## Getting Started

1. Log in to your Shopify admin and navigate to **Online Store → Themes**
2. Click the three dots next to your active theme and select **Edit Code**
3. Locate the header or navigation Liquid/JavaScript file in the code editor
4. Add the provided CSS/JS snippet to enable hover-triggered dropdowns
5. Save the file and verify the behavior on your storefront

> **Tip:** Always duplicate your theme before making code changes.

## Customization

- **Hover delay:** Add a small CSS transition delay to prevent accidental triggers
- **Animation:** Configure the dropdown open/close animation style
- **Mobile behavior:** Ensure the hover logic only applies to non-touch devices using CSS media queries or a JS touch detection check
