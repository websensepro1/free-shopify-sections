# Collection Carousel Slider Section

A free Collection Carousel Slider section for Shopify themes that you can use on any theme, free or paid, to display your collections in a smooth, scrollable carousel.

## Overview

The Collection Carousel Slider section presents your Shopify collections in an interactive horizontal carousel. Customers can browse through collection thumbnails with smooth sliding animations, encouraging exploration of your catalog. This section works out of the box with any Shopify theme and requires no additional apps.

## Features

- Horizontal scrolling collection carousel
- Smooth slide animations and transitions
- Compatible with any Shopify theme (Free or Paid)
- Configurable number of visible slides
- Navigation arrows and optional dot indicators
- Auto-play support with adjustable speed
- Fully responsive for mobile and desktop
- No paid app required

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Collection Carousel Slider** section to your desired page template
4. Select the collections to display in the carousel
5. Save and publish your changes

## Customization

- **Collections:** Choose which collections appear in the carousel
- **Slides per view:** Configure how many collection cards are visible at once
- **Auto-play:** Enable/disable automatic advancement and set the interval
- **Navigation:** Show or hide previous/next arrows and dot indicators
- **Card layout:** Adjust image aspect ratio, title visibility, and card spacing
- **Section heading:** Add an optional title above the carousel
