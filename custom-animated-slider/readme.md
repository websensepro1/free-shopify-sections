# Custom Animated Slider

Add a custom animated image slider to your Shopify store without any app, featuring smooth transitions and full customization through the theme editor.

## Overview

The Custom Animated Slider section provides a fully featured, app-free slideshow for your Shopify storefront. Built entirely with Liquid and CSS/JavaScript, it supports animated transitions, overlay content, and navigation controls — ideal for hero banners, promotions, or featured product highlights.

## Features

- Smooth animated slide transitions
- No app required — pure Liquid and CSS/JS implementation
- Works on any Shopify theme (Free or Paid)
- Auto-play with configurable interval
- Navigation arrows and dot indicators
- Per-slide overlay text, heading, and CTA button
- Fully responsive for mobile and desktop
- Easy management through the Shopify theme customizer

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Custom Animated Slider** section to your desired page template
4. Add slide blocks — each with an image, heading, subtext, and CTA button
5. Save and publish your changes

## Customization

- **Slides:** Add, remove, or reorder slides via the theme customizer
- **Auto-play:** Enable automatic advancement and configure the interval
- **Animation type:** Choose from fade, slide, or zoom transitions
- **Overlay content:** Set heading, subtext, and CTA button per slide
- **Navigation:** Show or hide arrows and progress dots
- **Slider height:** Configure height for desktop and mobile breakpoints
