# Add Double Scrolling Text

Easily add a custom double scrolling marquee effect to your Shopify store using simple Liquid code, creating an eye-catching announcement or promotional banner.

## Overview

The Double Scrolling Text section adds two synchronized marquee-style text strips to your Shopify storefront. This effect is perfect for announcements, promotions, brand slogans, or any repeating message you want to continuously scroll across the page. The implementation uses clean Liquid and CSS with no JavaScript dependencies.

## Features

- Two scrolling text rows for visual depth
- Smooth, continuous CSS marquee animation
- Fully customizable text content and speed
- Responsive design that works on all devices
- Lightweight — pure CSS, no JavaScript required
- Easy to integrate into any Shopify theme
- Supports custom fonts and colors

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Double Scrolling Text** section to your desired page template
4. Enter your desired text content for each scrolling row
5. Save and publish your changes

## Customization

- **Text content:** Set unique messages for each of the two scrolling rows
- **Scroll speed:** Adjust the CSS animation duration to control how fast the text moves
- **Direction:** Configure rows to scroll in opposite directions for added visual interest
- **Font & colors:** Customize typography and background to match your brand
- **Spacing:** Control padding and row height to fit your layout
