# Free 3D Product & Collection Slider for Shopify

Add a stunning 3D product and collection slider section to any Shopify theme — free or paid — with no app required.

## Overview

This section gives your Shopify storefront a modern, interactive 3D slider to showcase products or collections. It integrates seamlessly with any theme and is fully customizable through the Shopify theme editor.

## Features

- Works with any Shopify theme (free or paid)
- No third-party apps required
- 3D animated slider effect for products and collections
- Configurable via the Shopify theme customizer
- Lightweight and performance-friendly

## Getting Started

1. Download or copy the section file from this directory
2. Go to your Shopify Admin → **Online Store → Themes**
3. Click **Edit Code** on your active theme
4. Under **Sections**, click **Add a new section** and paste the code
5. Add the section to your desired page via the theme customizer

## Customization

Once added, the slider can be configured directly in the **Shopify Theme Editor**:
- Select products or collections to display
- Adjust slider speed and transition style
- Set the number of visible slides
