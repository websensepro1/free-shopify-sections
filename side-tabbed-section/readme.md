# Side Tabbed Section

Add a Side Tabbed Section to your Shopify store without any app, organizing content into a clean tabbed layout with vertical side navigation.

## Overview

The Side Tabbed Section presents content in a two-panel layout where vertical tabs on the left let customers switch between different content panels on the right. This is ideal for displaying product details, FAQs, shipping information, size guides, or any categorized content in a clean, organized format.

## Features

- Vertical side navigation with tabbed content panels
- Smooth tab switching without page reload
- No app required — built with Liquid and CSS/JavaScript
- Works on any Shopify theme (Free or Paid)
- Add unlimited tabs with custom content per tab
- Fully responsive — collapses to accordion on mobile
- Easy to configure via the Shopify theme customizer

## Getting Started

1. In your Shopify admin, navigate to **Online Store → Themes → Edit Code**
2. Under `sections/`, create a new section named `side-tabbed-section.liquid`
3. Paste the provided Liquid code into the new file and save
4. Navigate to **Themes → Customize** and add the **Side Tabbed Section** to your desired template
5. Add tab blocks with titles and content
6. Save and publish your changes

## Customization

- **Tabs:** Add, remove, or reorder tab blocks via the theme customizer
- **Tab labels:** Set a title for each tab that appears in the side navigation
- **Tab content:** Add rich text, images, or HTML content to each tab panel
- **Active tab:** Configure which tab is selected by default
- **Styling:** Adjust tab colors, fonts, and active state indicators
