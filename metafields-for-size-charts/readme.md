# Metafields for Size Charts

Add product-specific size chart popups to your Shopify Horizon theme for free, without using any apps, by leveraging Shopify metafields.

## Overview

The Metafields for Size Charts section uses Shopify metafields to store and display unique size chart data per product. A popup or inline display shows the relevant size chart when a customer views the product page, helping them choose the right size and reducing return rates — all without a paid app.

## Features

- Product-specific size charts using Shopify metafields
- Popup or inline size chart display on product pages
- Compatible with Shopify Horizon theme (and adaptable to others)
- No paid app required
- Each product can have its own unique size chart
- Easy to manage via Shopify admin metafield editor
- Fully responsive table layout
- Reduces returns by helping customers choose the correct size

## Getting Started

1. In your Shopify admin, navigate to **Settings → Custom data → Products** and create a metafield definition for size chart data
2. Copy the section/snippet Liquid file into your theme's appropriate directory
3. Add the size chart snippet to your product template Liquid file
4. In each product's admin page, populate the size chart metafield with your chart data
5. Save and publish your changes

## Customization

- **Metafield type:** Use a rich text, JSON, or table metafield to store size data
- **Display style:** Choose between a popup modal or inline table
- **Trigger:** Configure the button text and placement (e.g., "Size Guide" near the variant selector)
- **Table styling:** Adjust column headers, row colors, and typography to match your brand
- **Per-product charts:** Each product can have a completely different size chart
