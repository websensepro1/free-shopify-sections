# Simple Testimonial Section

A free Simple Testimonial Section for Shopify themes that you can use on any theme, free or paid, to display customer reviews and build social proof.

## Overview

The Simple Testimonial Section adds a clean, professional customer review display to your Shopify storefront. Showcase testimonials from satisfied customers to build trust, increase credibility, and encourage new visitors to make a purchase. The section is lightweight, fully customizable, and compatible with any Shopify theme.

## Features

- Clean, simple testimonial card layout
- Compatible with any Shopify theme (Free or Paid)
- No paid app required
- Add unlimited testimonial blocks
- Display customer name, photo, quote, and star rating
- Responsive grid or carousel layout options
- Easy to manage via the Shopify theme customizer

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Simple Testimonial Section** to your desired page template
4. Add testimonial blocks with customer name, quote, rating, and optional photo
5. Save and publish your changes

## Customization

- **Testimonials:** Add, remove, or reorder customer quote blocks
- **Customer name & title:** Set the reviewer's name and optional role or location
- **Quote:** Enter the customer testimonial text
- **Star rating:** Configure a 1–5 star rating per testimonial
- **Photo:** Upload an optional customer avatar image
- **Layout:** Choose between a grid, single column, or carousel display style
- **Section heading:** Add an optional title above the testimonials
