# Add Pricing Plan

Step-by-step guide to building a stylish pricing plan section with a comparison table for your Shopify theme, helping customers clearly understand your offerings.

## Overview

The Pricing Plan section allows you to display tiered pricing options with a feature comparison table directly on your Shopify storefront. Whether you offer subscription plans, service packages, or product bundles, this section provides a clean, professional layout that helps customers make informed purchase decisions.

## Features

- Multi-tier pricing plan display
- Feature comparison table layout
- Highlight a recommended or most popular plan
- Fully customizable plan names, prices, and features
- Responsive design for all screen sizes
- Easy to integrate into any Shopify theme (Free or Paid)
- Built with Liquid, CSS, and minimal JavaScript

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Pricing Plan** section to your desired page template
4. Configure your plan tiers, prices, and feature list
5. Save and publish your changes

## Customization

- **Plan names & prices:** Set unique names, pricing, and billing periods for each tier
- **Feature list:** Add or remove comparison rows to match your offerings
- **Highlighted plan:** Mark one plan as featured or recommended
- **Colors & typography:** Adjust to match your store's brand
- **CTA buttons:** Customize button text and links for each plan
