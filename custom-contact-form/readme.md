# Custom Contact Form

A step-by-step guide for Shopify store owners to build and customize their contact forms for free, without relying on paid apps.

## Overview

The Custom Contact Form section gives you full control over your Shopify store's contact page. Add custom fields, adjust the layout, and style the form to match your brand — all through Liquid and the Shopify theme customizer. This replaces the default contact form with a more tailored and professional experience.

## Features

- Fully customizable contact form fields
- No paid app required
- Works on any Shopify theme (Free or Paid)
- Add, remove, or reorder form fields
- Custom labels, placeholders, and validation
- Styled submit button with configurable text
- Responsive layout for all devices
- Integrates with Shopify's native form submission handling

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Custom Contact Form** section to your Contact page template
4. Configure form fields, labels, and submit button text
5. Save and publish your changes

## Customization

- **Form fields:** Add text inputs, email, phone, dropdowns, or textarea fields
- **Labels & placeholders:** Customize each field's label and placeholder text
- **Required fields:** Mark specific fields as required
- **Submit button:** Configure button label, color, and style
- **Success message:** Set the confirmation message shown after submission
- **Layout:** Adjust form width, padding, and alignment
