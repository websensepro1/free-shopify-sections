# Shining Hover Effect on Buy Button

Learn how to add a shining hover effect to the Buy Button in your Shopify store, enhancing the call-to-action with an eye-catching CSS animation that draws attention to the purchase button.

## Overview

This guide walks you through adding a polished shining/gloss sweep animation to the Add to Cart or Buy Now button in your Shopify theme. The effect triggers on hover, creating a light sweep that makes the button more visually dynamic and encourages clicks — improving conversion rates without any paid apps.

## Features

- Shining light sweep animation on hover
- Applied directly to the buy/add-to-cart button
- No paid app required — pure CSS implementation
- Compatible with Shopify Dawn theme (v15.1.0+)
- Subtle, professional effect that enhances the CTA
- Easy to implement in a few steps
- Can be adapted to other themes and buttons

## Getting Started

1. Log in to your Shopify admin and navigate to **Online Store → Themes**
2. If not already using Dawn, click **Add Theme** to install the latest version and publish it
3. Click the three dots next to your active theme and select **Edit Code**
4. Locate the `buy-buttons.liquid` file in the code editor
5. Add the provided CSS code for the shining hover effect
6. Save the file and verify the effect on your storefront

> **Tip:** If working on a live theme, duplicate it first before making changes to avoid unintended issues.

## Customization

- **Animation speed:** Adjust the CSS transition duration to make the shine faster or slower
- **Shine color:** Modify the gradient color and opacity for a subtle or bold effect
- **Trigger:** Apply the same CSS to other buttons (newsletter, checkout, etc.)
- **Angle:** Change the gradient angle to alter the direction of the light sweep
