# Autoplay Video Section

Add an autoplay video section to your Shopify store by creating a new section file and pasting in the provided Liquid code for an instant, engaging video display.

## Overview

The Autoplay Video Section embeds a background or featured video that plays automatically when customers visit your page. This is ideal for product demonstrations, brand storytelling, or creating an immersive hero experience. The section is built with a straightforward Liquid implementation that works across all Shopify themes.

## Features

- Autoplay video playback on page load
- Supports YouTube, Vimeo, or hosted video files
- Muted autoplay for browser compatibility
- Optional overlay text and CTA button
- Fully responsive for all screen sizes
- Compatible with any Shopify theme
- Simple setup with a single Liquid file

## Getting Started

1. In your Shopify admin, navigate to **Online Store → Themes → Edit Code**
2. Under `sections/`, click **Add a new section** and name it `autoplay-video-section`
3. Paste the provided Liquid code into the new file and save
4. Navigate to **Themes → Customize** and add the **Autoplay Video Section** to your desired template
5. Configure the video URL and any overlay content
6. Save and publish your changes

## Customization

- **Video source:** Enter your YouTube, Vimeo, or direct video file URL
- **Overlay text:** Add a heading and subtext displayed on top of the video
- **CTA button:** Configure button text and destination link
- **Autoplay & loop:** Toggle autoplay and loop behavior
- **Fallback image:** Set a poster image shown before the video loads
- **Section height:** Adjust the video container height for different devices
