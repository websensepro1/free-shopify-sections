## 🚀 Installation

### Step 1: Download the Code
1. Navigate to the section folder you want to use.
2. Copy the `.liquid` file contents.

### Step 2: Add to Your Shopify Theme
1. Go to **Shopify Admin → Online Store → Themes**.
2. Click the **⋯** menu next to your active theme and select **Edit code**.
3. Under **Sections**, click **Add a new section**.
4. Name it as instructed in the section's readme, paste the code, and click **Save**.

### Step 3: Activate in the Theme Customizer
1. Go back to **Online Store → Themes** and click **Customize**.
2. Navigate to the page where you want the section.
3. Click **+ Add Section**, find your section by name, and configure it.
