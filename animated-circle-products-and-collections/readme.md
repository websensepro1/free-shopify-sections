# Animated Circle Products and Collections

Enhance your Shopify store's UI by adding a stylish animated circle product and collection display that creates an engaging, modern browsing experience.

## Overview

The Animated Circle Products and Collections section presents your products or collections in circular frames with smooth CSS animations. This visually distinctive layout is ideal for highlighting featured items, new arrivals, or key collections, encouraging customers to explore your catalog in a fresh and interactive way.

## Features

- Animated circular product and collection display
- Smooth CSS hover and entrance animations
- Supports both products and collections
- Configurable number of items per row
- Fully responsive for all screen sizes
- Compatible with any Shopify theme
- Easy setup via the Shopify theme customizer

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Animated Circle Products and Collections** section to your desired page template
4. Select or configure which products or collections to display
5. Save and publish your changes

## Customization

- **Content source:** Choose to display products, collections, or a mix of both
- **Item count:** Set how many circles appear in the section
- **Animation style:** Adjust entrance animation type and duration
- **Circle size:** Configure the diameter for desktop and mobile views
- **Labels:** Show or hide product/collection names beneath each circle
- **Link behavior:** Each circle links to its respective product or collection page
