# Lookbook Section

Add a Lookbook section to your Shopify store without an app, compatible with Dawn v14 and other modern themes, to create shoppable editorial image layouts.

## Overview

The Lookbook section lets you create styled editorial layouts where customers can discover and shop products directly from inspirational images. Hotspots or overlays on the lookbook image link directly to product pages, creating an immersive, magazine-style shopping experience — all without a paid app.

## Features

- Shoppable lookbook image layout with product hotspots
- Compatible with Shopify Dawn v14 and any theme (Free or Paid)
- No app required — built with Liquid and CSS
- Add clickable product tags overlaid on lookbook images
- Fully responsive for mobile and desktop
- Easy to manage via the Shopify theme customizer
- Clean, editorial aesthetic

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Lookbook** section to your desired page template
4. Upload your lookbook image and add product hotspot blocks
5. Position each hotspot over the relevant product in the image
6. Save and publish your changes

## Customization

- **Lookbook image:** Upload a high-resolution editorial or lifestyle photo
- **Product hotspots:** Add product tag overlays with position coordinates
- **Linked products:** Connect each hotspot to a product page
- **Hotspot style:** Customize the appearance of the tag indicator
- **Layout:** Adjust image sizing and content area alignment
- **Section heading:** Add an optional title above the lookbook
