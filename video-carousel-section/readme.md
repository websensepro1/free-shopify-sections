# Video Carousel Section

A free Video Carousel Section for Shopify themes that you can use on any theme, free or paid, to display multiple videos in an interactive scrollable carousel.

## Overview

The Video Carousel Section embeds a horizontally scrollable carousel of videos on your Shopify storefront. Use it to showcase product demonstrations, brand films, tutorials, or customer testimonial videos. The section supports YouTube and Vimeo embeds and works across all Shopify themes without a paid app.

## Features

- Horizontal scrolling video carousel
- Supports YouTube and Vimeo embeds
- Compatible with any Shopify theme (Free or Paid)
- No paid app required
- Navigation arrows and optional dot indicators
- Auto-play, mute, and loop controls per video
- Configurable number of visible videos
- Fully responsive for all screen sizes

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Video Carousel Section** to your desired page template
4. Add video blocks — each with a YouTube or Vimeo URL and optional title
5. Save and publish your changes

## Customization

- **Videos:** Add, remove, or reorder video blocks via the theme customizer
- **Video source:** Enter a YouTube or Vimeo URL for each video
- **Video title:** Set an optional caption displayed below each video
- **Slides per view:** Configure how many videos are visible at once
- **Navigation:** Show or hide previous/next arrows and dot indicators
- **Section heading:** Add an optional title above the carousel
