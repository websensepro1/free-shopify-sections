# Sales Popup

Add a recent sales popup to your Shopify store for free without using any apps, creating social proof notifications that encourage purchases.

## Overview

The Sales Popup section displays real-time or simulated recent purchase notifications to store visitors. These social proof popups show messages like "John from New York just bought [Product]" and are proven to increase conversion rates by creating urgency and demonstrating that others are actively purchasing from your store — all without a paid app.

## Features

- Recent sales / social proof popup notifications
- No app required — built with Liquid and JavaScript
- Works on any Shopify theme (Free or Paid)
- Configurable popup content, timing, and frequency
- Smooth entrance and exit animations
- Mobile-friendly popup positioning
- Customizable appearance to match your brand
- Fully controllable via the theme customizer

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Sales Popup** section to your theme layout (Global or specific pages)
4. Configure popup messages, timing, and display settings
5. Save and publish your changes

## Customization

- **Popup messages:** Configure the product names, buyer names, and locations shown
- **Display delay:** Set how long after page load before the first popup appears
- **Display interval:** Control how frequently new popups appear
- **Auto-dismiss:** Set the duration a popup stays visible before closing
- **Position:** Configure popup placement (bottom-left, bottom-right, etc.)
- **Styling:** Adjust colors, fonts, and popup size to match your brand
