# Vertical Card Slider

Add a Vertical Card Slider to your Shopify store featuring smooth GSAP animations, creating a dynamic, eye-catching product or content display that scrolls vertically.

## Overview

The Vertical Card Slider section presents cards in a vertical scrolling layout powered by GSAP animations. This unique orientation is ideal for showcasing featured products, testimonials, blog posts, or promotional content in a way that stands out from the standard horizontal carousel — giving your storefront a modern, polished feel.

## Features

- Vertical scrolling card slider with GSAP animations
- Smooth, fluid animation on scroll or auto-advance
- Compatible with any Shopify theme (Free or Paid)
- No paid app required
- Configurable card content (image, title, description, link)
- Fully responsive for mobile and desktop
- Easy to configure via the Shopify theme customizer

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Vertical Card Slider** section to your desired page template
4. Add card blocks with image, title, description, and link
5. Save and publish your changes

## Customization

- **Cards:** Add, remove, or reorder card blocks via the theme customizer
- **Card content:** Set an image, title, description, and optional CTA link per card
- **Animation speed:** Adjust the GSAP animation duration and easing
- **Auto-play:** Enable or disable automatic vertical advancement
- **Number of visible cards:** Configure how many cards are visible at once
- **Section heading:** Add an optional title above the slider
