# Animated Image Slider

Learn how to create an animated image slider in Shopify without an app, giving your store a polished, interactive visual experience.

## Overview

The Animated Image Slider section provides a fully functional, app-free image slideshow for your Shopify store. Built with Liquid and CSS animations, it supports automatic playback, navigation controls, and smooth transitions — perfect for showcasing promotions, featured products, or brand storytelling.

## Features

- Smooth animated slide transitions
- Auto-play with configurable interval
- Previous/next navigation controls
- Optional dot indicators
- Supports images with overlay text and CTA buttons
- Fully responsive for all screen sizes
- No app required — pure Liquid and CSS

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Animated Image Slider** section to your desired page template
4. Upload your slide images and configure each slide's text and links
5. Save and publish your changes

## Customization

- **Slides:** Add, remove, or reorder slides in the theme customizer
- **Auto-play speed:** Set the interval between automatic slide transitions
- **Animation type:** Choose from fade, slide, or other transition styles
- **Overlay text:** Configure heading, subtext, and CTA button per slide
- **Navigation:** Show or hide arrows and dot indicators
- **Height:** Set the slider height for desktop and mobile
