# Add Custom Input Fields to Shopify Product Page

Enhance your Shopify store by adding custom input fields to your product pages — enabling personalization options like engraving text, gift messages, custom sizes, and more.

## Overview

This guide walks you through adding custom fields to the Shopify product page without requiring a paid app. The solution uses Liquid templating and a small snippet of JavaScript to capture and pass custom input data along with the cart item.

## Features

- No paid apps required
- Works with any Shopify theme
- Captures custom input at add-to-cart time
- Data persists in cart line item properties

## Getting Started

1. Navigate to your Shopify Admin → **Online Store → Themes**
2. Click **Edit Code** on your active theme
3. Follow the steps in the video to add the custom field snippet
