# Icons Slider

An Icon Slider section for Shopify that works with all themes, allowing you to easily add a scrolling icon/logo slider without any additional paid apps.

## Overview

The Icons Slider section adds a smooth, continuous scrolling strip of icons, logos, or brand marks to your Shopify storefront. It is perfect for displaying partner logos, brand affiliations, payment badges, trust icons, or featured-in press mentions — all without requiring a paid app.

## Features

- Smooth scrolling icon/logo carousel
- Compatible with all Shopify themes (Free or Paid)
- No paid app required
- Configurable scroll speed and direction
- Upload custom SVG, PNG, or JPG icons
- Optional links per icon
- Fully responsive for all screen sizes
- Lightweight CSS-only or minimal JS implementation

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Icons Slider** section to your desired page template
4. Add icon blocks — each with an image and optional link
5. Save and publish your changes

## Customization

- **Icons:** Upload custom icon or logo images for each block
- **Links:** Optionally link each icon to a URL
- **Scroll speed:** Adjust the CSS animation duration to control scrolling pace
- **Direction:** Configure left-to-right or right-to-left scrolling
- **Icon size:** Set the height and spacing for icons
- **Pause on hover:** Enable or disable pausing when a user hovers over the slider
