# WhatsApp Chat Icon

Add a floating WhatsApp Chat Icon to your Shopify store, allowing customers to instantly reach you on WhatsApp directly from any page on your storefront.

## Overview

The WhatsApp Chat Icon section adds a fixed floating button to your Shopify storefront that opens a WhatsApp chat with your business when clicked. This is an effective way to offer real-time customer support, reduce friction in the buying process, and increase conversions by making it effortless for customers to ask questions.

## Features

- Floating WhatsApp chat button fixed to the screen
- Tapping/clicking opens a WhatsApp chat instantly
- Compatible with any Shopify theme (Free or Paid)
- No paid app required
- Configurable phone number and pre-filled message
- Fully responsive — works on mobile and desktop
- Smooth appearance animation
- Lightweight, minimal code footprint

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **WhatsApp Chat Icon** section to your theme layout (Global/Footer area)
4. Enter your WhatsApp phone number (with country code) and an optional pre-filled greeting message
5. Save and publish your changes

## Customization

- **Phone number:** Enter your WhatsApp business number in international format (e.g., +1234567890)
- **Pre-filled message:** Set a default message that populates when a customer opens the chat
- **Position:** Configure the button position (bottom-right or bottom-left)
- **Icon size:** Adjust the floating button size
- **Visibility:** Show the button on all pages or restrict to specific templates
- **Delay:** Add an appearance delay so the button shows after a few seconds on page load
