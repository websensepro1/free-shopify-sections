# Scrolling Images Slider

Add a smooth, continuously scrolling images slider to your Shopify store to create an engaging visual marquee effect for product photos, brand logos, or promotional imagery.

## Overview

The Scrolling Images Slider section creates an infinite, auto-scrolling strip of images on your Shopify storefront. Unlike a standard carousel, this section scrolls continuously without stopping, making it ideal for partner logos, brand imagery, product highlights, or any collection of images you want to display in a flowing, eye-catching format.

## Features

- Smooth, infinite continuous scrolling
- No pagination or navigation buttons needed
- Compatible with any Shopify theme (Free or Paid)
- No paid app required
- Configurable scroll speed and direction
- Upload any number of images
- Optional links per image
- Fully responsive for all screen sizes

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Scrolling Images Slider** section to your desired page template
4. Upload your images and configure optional links for each
5. Save and publish your changes

## Customization

- **Images:** Add or remove image blocks via the theme customizer
- **Scroll speed:** Adjust the CSS animation duration to control how fast images scroll
- **Direction:** Set left-to-right or right-to-left scrolling
- **Image size:** Configure image height and aspect ratio
- **Gap:** Adjust spacing between images
- **Pause on hover:** Enable or disable scroll pausing when the user hovers
