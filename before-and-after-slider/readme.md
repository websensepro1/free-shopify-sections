# Before and After Slider

Add a Before and After image comparison slider to your Shopify store without any app — updated implementation using pure Liquid and CSS.

## Overview

The Before and After Slider section lets customers drag a divider to compare two images side by side. This is especially powerful for stores selling products with transformative results, such as skincare, cleaning products, home improvement, or fitness items. The implementation requires no paid apps and works on any Shopify theme.

## Features

- Interactive drag-to-compare image slider
- No app required — built with Liquid and CSS/JavaScript
- Works on any Shopify theme (Free or Paid)
- Touch and mouse drag support for mobile and desktop
- Configurable before and after labels
- Fully responsive design
- Easy one-file installation

## Getting Started

1. In your Shopify admin, navigate to **Online Store → Themes → Edit Code**
2. Under `sections/`, click **Add a new section** and name it `before-after-slider`
3. Paste the provided Liquid code into the new file and save
4. Navigate to **Themes → Customize** and add the **Before and After Slider** section to your desired template
5. Upload your before and after images
6. Save and publish your changes

## Customization

- **Before image:** Upload the starting state image
- **After image:** Upload the resulting state image
- **Labels:** Set custom text for the "Before" and "After" labels
- **Starting position:** Configure the initial divider position (e.g., 50%)
- **Orientation:** Horizontal or vertical slider direction
- **Section heading:** Add an optional title above the slider
