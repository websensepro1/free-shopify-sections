# Stylish Product Grid Section

Add a stylish and interactive product grid section to your Shopify store without using any apps or complex coding, creating an engaging product browsing experience.

## Overview

The Stylish Product Grid Section provides a visually enhanced, interactive product listing layout for your Shopify storefront. With hover effects, clean card designs, and flexible grid options, this section elevates the standard product collection display into a polished, conversion-focused experience — all without a paid app.

## Features

- Stylish product card design with hover effects
- No app required — built with Liquid and CSS
- Compatible with any Shopify theme (Free or Paid)
- Configurable grid columns (2, 3, 4, or 5 per row)
- Displays product image, title, price, and sale badge
- Quick add-to-cart button on hover
- Fully responsive for mobile and desktop
- Easy to configure via the Shopify theme customizer

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Stylish Product Grid** section to your desired page or collection template
4. Select the collection or products to display
5. Configure the number of columns and items per page
6. Save and publish your changes

## Customization

- **Collection source:** Choose a collection or manually select featured products
- **Grid columns:** Set the number of product columns per row
- **Items per page:** Control how many products are shown before pagination
- **Hover effect:** Customize the card hover animation and quick-add behavior
- **Image ratio:** Set square, portrait, or landscape image aspect ratio
- **Sale badge:** Show or hide the sale/discount badge on product cards
