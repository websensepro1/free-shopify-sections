# Animated Team Section

A free Animated Team Section for Shopify themes that you can use on any theme, free or paid, to introduce your team members with style.

## Overview

The Animated Team Section lets you showcase your team members on your Shopify storefront with smooth CSS animations. Each team member card displays their photo, name, role, and optional social links, bringing a human touch to your brand and building customer trust.

## Features

- Animated team member cards with hover effects
- Display photo, name, job title, and bio for each member
- Optional social media links per team member
- Compatible with any Shopify theme (Free or Paid)
- Fully responsive grid layout
- Easy to manage through the Shopify theme customizer
- No external dependencies required

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Animated Team Section** to your desired page template
4. Add team members using the block settings (photo, name, title, bio)
5. Save and publish your changes

## Customization

- **Team members:** Add or remove member blocks via the theme customizer
- **Photos:** Upload a profile image for each team member
- **Name & title:** Set each member's display name and job title
- **Bio:** Add a short description for each team member
- **Social links:** Configure optional links to social media profiles
- **Columns:** Adjust the number of members displayed per row
- **Animation:** Customize entrance animation style and timing
