# Breadcrumb Navigation — Shopify Section

A fully customizable breadcrumb navigation section for Shopify themes. Supports product pages, collection pages, article pages, and nested menu structures up to 3 levels deep.

## Video Tutorial

[![Breadcrumb Navigation Tutorial](https://img.youtube.com/vi/xue6ePgO02Y/maxresdefault.jpg)](https://youtu.be/xue6ePgO02Y)

## Features

- Breadcrumbs for **product**, **collection**, and **article** pages
- Supports **nested navigation up to 3 levels deep** (built from your main-menu linklist)
- **JSON-LD BreadcrumbList schema** output for SEO structured data
- Optional **home icon** (SVG house icon) next to the Home link
- 8 **delimiter style options**: Angle Right, Slash, Arrow Right, Squiggle Arrow, Right Long, Double Right, Diamond Arrow Head, Heavy Angle Right
- Customizable **background color** and **text color**
- Optional **accent color** for the current (visited) page link
- Granular visibility controls: enable per page type or site-wide
- The last breadcrumb link is automatically made non-navigable (points to `javascript:void(0)`)

## Installation

1. In your Shopify admin, go to **Online Store → Themes → Edit code**
2. Under the `sections/` folder, click **Add a new section**
3. Name it `breadcrumbs` and paste the contents of `breadcrumbs.liquid`
4. Save the file

To add the section to a page:
- Go to **Online Store → Themes → Customize**
- Navigate to a product or collection page
- Click **Add section** and select **Breadcrumb Navigation**

## Schema Settings

| Setting | Type | Default | Description |
|---|---|---|---|
| Use home icon | Checkbox | `true` | Shows an SVG house icon before the Home link |
| Delimiter Icon | Select | `angle_right` | The separator character between breadcrumb items |
| Background Color | Color | `#fff` | Background color of the breadcrumb bar |
| Breadcrumb Color | Color | `#465076` | Text and link color |
| Enable visited color | Checkbox | `false` | Highlights the current page link in an accent color |
| Visited Color | Color | `#4770db` | Accent color for the current page link |
| Enable in Product Page | Checkbox | — | Show breadcrumbs only on product pages |
| Enable in Collection Page | Checkbox | — | Show breadcrumbs only on collection pages |
| Enable on Complete Site | Checkbox | `true` | Show breadcrumbs on all page types |

### Delimiter Options

| Value | Symbol |
|---|---|
| Angle Right | `›` |
| Slash | `/` |
| Arrow Right | `➤` |
| Squiggle Arrow | `⟿` |
| Right Long | `➞` |
| Double Right | `»` |
| Diamond Arrow Head | `⤞` |
| Heavy Angle Right | `❯` |

## Page Type Behavior

| Template | Breadcrumb Path |
|---|---|
| Product | Home › [Menu L1] › [Menu L2] › [Menu L3] › Product Title |
| Collection | Home › Collection Title |
| Article | Home › [Blog Menu L1] › [Blog Menu L2] › [Blog Menu L3] › Article Title |
| Index / Cart | Hidden |

## Notes

- Breadcrumbs are hidden by default on `index` (homepage) and `cart` templates
- The product and article breadcrumb paths are resolved by walking your `main-menu` linklist hierarchy up to 3 levels deep — no manual configuration needed
- JSON-LD schema is output for product, collection, and article pages automatically, which helps Google display breadcrumbs in search results
- The section uses inline `<style>` generated by Liquid so color settings apply without an external stylesheet
